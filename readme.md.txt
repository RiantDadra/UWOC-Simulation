# UWOC System Simulation

## Overview
Simulation of Underwater Wireless Optical Communication (UWOC) systems using OOK and 4-QAM modulation. The system models underwater optical signal propagation using the Beer-Lambert law and evaluates BER performance under different channel conditions.

## Features
- Beer-Lambert attenuation model  
- AWGN channel modeling  
- BER vs SNR analysis  
- OOK vs 4-QAM performance comparison
