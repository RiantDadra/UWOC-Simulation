# UWOC System Simulation (OOK & 4-QAM)

## Overview
This project presents a simulation-based analysis of Underwater Wireless Optical Communication (UWOC) systems.

## Key Features
- OOK and 4-QAM modulation
- Beer-Lambert attenuation model
- AWGN channel
- Spatial and Frequency Diversity
- BER vs SNR and Distance analysis

## System Model
Transmitter → Channel → Receiver:
- Modulation: OOK / 4-QAM  
- Channel: Attenuation + Noise  
- Receiver: Detection + BER  

## Results

### BER vs SNR
Implemented and plotted various simulations related to BER, SNR and Channel Characteristics
